# Quicklinks Reference

URLs and apps you can open instantly with a keyword.

## Local Development

| Name | Link | Keyword |
|------|------|---------|
| Grafana | `http://localhost:3000` | `grafana` |
| Prometheus | `http://localhost:9090` | `prom` |
| Local API | `http://localhost:8080` | `api` |

## Enterprise Services

Customize these with your organization's URLs:

| Name | Link | Keyword |
|------|------|---------|
| Enterprise GitHub | `https://github.example.com` | `ghent` |

## Search URLs (Dynamic)

Use `{query}` placeholder for search input.

| Name | Link | Keyword |
|------|------|---------|
| GitHub Search | `https://github.com/search?q={query}` | `ghs` |
| NPM Search | `https://www.npmjs.com/search?q={query}` | `npm` |
| Stack Overflow | `https://stackoverflow.com/search?q={query}` | `so` |
| MDN | `https://developer.mozilla.org/en-US/search?q={query}` | `mdn` |
| Crates.io | `https://crates.io/search?q={query}` | `crate` |
| PyPI | `https://pypi.org/search/?q={query}` | `pypi` |

## How to Add

1. Open Raycast
2. Search "Create Quicklink"
3. Enter Name, Link, and optional Keyword
4. For search URLs, use `{query}` where the search term goes
